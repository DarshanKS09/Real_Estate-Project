# 🏡 Real Estate Platform 

A full-stack real estate web application that allows users to browse, search, and manage property listings with authentication and modern UI.

---

## 🔥 Features

- 🔐 User Authentication (JWT + Cookies)
- 🏠 Property Listings (Buy / Rent)
- 🔍 Advanced Search & Filtering
- 🧑‍💼 Role-based Access (User / Agent)
- 📝 CRUD Operations for Properties
- 📸 Image Upload Support (Cloud Storage)
- 🔔 Notifications System
- 📱 Fully Responsive UI

---

## 🧠 What This Project Solves

This platform acts as a bridge between:
- Property owners / agents → list properties
- Users → search and explore properties easily

It simplifies the process of discovering and managing real estate listings in one place.

---

## 🛠️ Tech Stack

### Frontend
- React.js / Next.js
- Tailwind CSS
- Axios

### Backend
- Node.js
- Express.js

### Database
- MongoDB

### Authentication
- JWT (JSON Web Tokens)
- Cookies
